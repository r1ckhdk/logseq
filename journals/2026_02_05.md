- [[RNP - Administração de Sistemas Linux]]
	- **Módulo 2**: Usuários e Grupos
		- Grupos
			- Um usuário pode fazer parte de vários grupos
				- Grupo primário
					- grupo principal, qualquer arquivo ou processo criado pelo usuário terá esse grupo de permissão
				- Grupo secundário
					- outros grupos de permissão
			- Arquivo `/etc/group`
				- grupo:senha:GID:lista_de_usuarios
			- `groupadd` / `addgroup`
		- Usuários
			- Apenas usuários cadastrados podem acessar o sistema
			- Contas dos usuários ficam no arquivo /etc/passwd
			- Internamente o sistema reconhece o usuários através de um número inteiro chamado UID
			- Arquivo `/etc/passwd`
				- usuario:senha:UID:GID:GECOS:diretorio_home:interpretador
			- Arquivo `/etc/shadow`
				- username:password:last_changed:minimum:maximum:warn:inactive:expire
			- Arquivo `/etc/skel`
				- Contém o modelo de diretório home que será utilizado pelo sistema para a criação das novas contas de usuários
					- **Global**: /etc/profile, /etc/bashrc
					- **Local**: .bashrc, .profile
		- **Administrando contas de usuários**
			- Desabilitando contas
				- Trocar o shell por mensagem explicativa
				- Colocar qualquer caractere no início da senha criptografada (!)
				- Comandos: `usermod`, `passwd`, `chage`
			- Removendo contas
				- Comandos: `userdel` e `deluser`
		- **Administando grupos**
			- `groupmod`: modifica informações no arquivo /etc/group
				- nome, GID e senha (desuso)
			- Removendo grupos
				- `groupdel`
				- `userdel` também pode remover o grupo com a opção `-f`
					- Vai forçar a remoção do grupo primário do usuário
		- **Autenticação**
			- PAM
				- Camada intermerdiária entre a aplicação e a autenticação real
				- Permite implementar de uma política de senha complexa definindo quantidade de caracteres, letras maiúsculas e minúsculas, números etc
				- Possibilita consultar um dicionário para validar a força da senha
				- É possível limitar o acesso a diversos recursos do sistema operacional
				- Arquivo: `/etc/pam.conf`
				- Diretório: `/etc/pam.d`
				- Module types:
					- Auth
					- Account: autorização para uso de programas que utilizam login
					- Password: operações referentes a alteração de senha
					- Session: início e encerramento de sessões e ambientes de usuário
				- Control flags:
					- Required
					- Sufficient
					- Optional
					- Requisite
					- Include
					- Substack
